# Private_network
L'idea è creare una piccola rete privata tra macchine virtuali che possono comunicare solo tra di esse con un' unica porta di uscita che puo essere usata solo da una macchina alla volta e che può essere solo se attivata da un sarver centrale. 

In quanto rete ci deve essere un server centrale che controlla gli indirizzamenti,  l'attribuzione degli indirizzi IP e il gateway. Il server inoltre deve avere una cache per mantenere la cronologia di tutti i messaggi inviati tra le macchine e per e da il gateway.
